# 初始化第一个小程序



## 1 为何使用taro 开发小程序



 taro小程序开发框架由京东开源贡献，https://github.com/jdf2e/nutui 5.5K星。

 多端框架，支付宝、微信等小程序、H5等 共用一套源码。

组件丰富，开箱即用。



## 2  初始化第一个小程序 



### 2.1 安装依赖包

```

sudo cnpm install -g @tarojs/cli


```



### 2.2  初始化项目骨架



```
sudo taro init taroApp
```



### 2.3  编译打包

```
sudo cnpm run dev:weapp
sudo cnpm run build:weapp
```
